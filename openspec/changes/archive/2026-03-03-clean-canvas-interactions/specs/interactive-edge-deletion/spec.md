## MODIFIED Requirements

### Requirement: Edge Delete Control
The system SHALL support deleting relationship edges through a centralized contextual toolbar instead of direct on-canvas buttons.

#### Scenario: Selection Highlighting
- **WHEN** an edge is selected
- **THEN** it is visually highlighted to indicate focus, and its deletion action is available in the global toolbar.

## REMOVED Requirements

### Requirement: Edge Deletion Sync
**Reason**: Replaced by centralized deletion logic in the contextual toolbar and keyboard shortcuts.
**Migration**: Use the "Delete" button in the top-right toolbar or press Delete/Backspace when an edge is selected.
