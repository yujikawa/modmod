## ADDED Requirements

### Requirement: Visual Feedback for Active Interactions
The system SHALL provide immediate visual feedback when a node is being interacted with, specifically during a drag operation.

#### Scenario: Node Dragging Feedback
- **WHEN** a user is dragging a node (table or domain)
- **THEN** the node SHALL appear "elevated" by increasing its shadow depth and slightly increasing its scale (e.g., 1.02x)
