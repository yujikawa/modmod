## ADDED Requirements

### Requirement: マルチコマンド解析
システムは、入力されたテキストに基づき、リレーション作成命令かドメイン操作命令かを自動的に判別しなければならない (SHALL parse commands)。

#### Scenario: mvコマンドの認識
- **WHEN** ユーザーがパネルに `mv ` から始まる文字列を入力する
- **THEN** システムはそれをドメイン移動命令と解釈し、UI を移動モード（サジェスト対象をドメインに限定）に切り替える

### Requirement: 選択状態に基づいた操作
システムは、キャンバス上で選択されている複数のエンティティを対象としてコマンドを実行できなければならない (SHALL respect selection)。

#### Scenario: 選択中テーブルの移動
- **WHEN** ユーザーが 3 つのテーブルを選択した状態で `mv Sales` と入力して実行する
- **THEN** 選択されていた 3 つのテーブルすべてが `Sales` ドメインへ移動する

### Requirement: ワイルドカード一括移動
システムは、テーブル名の指定においてワイルドカード（`*`）をサポートし、条件に合致するエンティティを一括で操作できなければならない (SHALL support wildcard bulk move)。

#### Scenario: プレフィックス一致での移動
- **WHEN** ユーザーが `mv dim_* to Master` と入力して実行する
- **THEN** `dim_` で始まるすべてのテーブルが `Master` ドメインへ一括移動する
