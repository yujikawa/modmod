## Why

データモデリングにおける「整理（移動）」と「接続（エッジ作成）」は、エンジニアにとって異なるコンテキストの作業です。これらをサイドバーの一つのタブに詰め込むと、UI が複雑になり直感性が損なわれます。
本変更では、接続機能はサイドバーの「Connect」タブに集約しつつ、Modscape 全体への命令を司る独立した「Command Palette (Ctrl+K)」を導入します。これにより、VS Code のような爆速の操作体験を提供し、大規模なモデル管理を劇的に効率化します。

## What Changes

- **Command Palette の新設**: 画面中央にフローティング表示される、コマンド入力専用のインターフェースを導入します。
- **Ctrl+K ショートカット**: パレットを瞬時に呼び出すためのグローバルショートカットを追加します。
- **Connect タブの純粋化**: `QuickConnectTab` から `mv` などの整理コマンドを削除し、ER/Lineage の接続に特化させます。
- **一括移動 (Bulk Move) のコマンド化**: `mv <pattern> to <domain>` などのコマンドをパレット経由で実行可能にします。
- **複数選択との連動**: キャンバスで選択中のオブジェクトを対象にしたコマンド実行をサポートします。

## Capabilities

### New Capabilities
- `command-palette`: Modscape 全体への命令をテキストベースで実行できる独立したフローティングインターフェース。
- `bulk-entity-organization`: 複数のテーブルをドメイン間で一括移動・整理するエンジン。

### Modified Capabilities
- `quick-connector`: サイドバー内で ER/Lineage 接続に特化したシンプルなツールとして再定義。

## Impact

- **UI**: 画面中央に `CommandPalette` コンポーネントが追加されます。
- **UX**: 接続作業と整理作業が分離され、それぞれの作業に集中できる環境が整います。
